# Compression Becomes Authority — Vocabulary

The four-term public vocabulary for the planned Neutral Ambassador essay. Captured here so on-disk pointer mentions (NRAF, boundary-calculus-program-candidate) reference a real artifact rather than a memory-only ghost. This is not the essay. It is the glossary that an essay would draw on.

## Four terms

1. **Admissibility problem.** The problem of deciding when and how an AI-mediated or compressed output may enter an institutional decision chain with authority.

2. **Compression-authority laundering.** The failure mode where a lossy representation — proxy, summary, model output, score, administrative compression — acquires authority as if it were the underlying object it merely represents.

3. **Counterfeit legibility.** A representation looks sufficiently orderly, formatted, quantified, or institutionally fluent to be treated as standing evidence, despite lacking the substrate relation needed to justify that treatment.

4. **Memory poisoning.** A compressed or mistaken representation persists into future state — records, summaries, model memory, case files, dashboards, policy traces — and later re-enters the decision chain as inherited fact.

## Kernel mappings

The vocabulary names *where* failure happens in administrative AI; the kernels name *the refusal discipline* that addresses it. Each term routes to existing substrate:

- admissibility problem → SurfaceAuthorization (gate)
- compression-authority laundering → FiatAdmissibility family
- counterfeit legibility → CollapsedSurface + synthetic-justification failure mode
- memory poisoning → admissibility-decay family
