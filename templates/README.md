# Templates

Reusable templates for the papers repository.

## Contents

- **README.md-template** — Standard README structure for preprint directories (title, abstract, citation, related papers, keywords)
- **PLAIN-LANGUAGE-SUMMARY.md-template** — Reader-oriented translation contract used by every preprint; explains the practical question and contribution without introducing new paper claims, and points back to the canonical artifact.
- **metadata.yaml-template** — Standard metadata fields for preprint papers (title, author, date, DOI, keywords, abstract, series info)

The README and plain-language summary are navigation surfaces. A plain-language
summary translates the paper's argument for discovery; it is not a compressed
section inventory or an ELI5 rewrite. Neither surface silently revises a
DOI-bearing paper.
