# Corpus Map

The conceptual order, not the chronological one. For the full inventory with status and DOIs, see **[Corpus](corpus.md)**; to pick an entry point by interest, see **[Start Here](start-here.md)**.

This is **one investigation** organized around a single question:

```
Admissibility — may this claim be used to authorize this consequence?
  ├─ temporal coherence / Δt        (the control-theoretic foundation)
  ├─ cybernetic & institutional failure
  ├─ social-epistemic collapse
  ├─ observer / witness integrity
  └─ formal kernels & governed software  (Lean / admissibility kernels)
```

A higher paper number is **newer, not stronger**. The threads below are entry points into the same investigation, not a reading order. Links go to each paper's source directory; grouping follows the [preprint README](https://github.com/unpingable/papers/blob/main/preprint/README.md).

## Superseded / narrowed

The corpus does not silently retire work. Where a claim was narrowed or broken by the formal pass, the move is recorded in the Lean claim register (STALE / BROKEN) rather than by deleting the paper. Historical formulations stay readable, marked by their register status — see [Methodology § Honest compounding](methodology.md).
