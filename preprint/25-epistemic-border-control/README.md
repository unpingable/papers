# Paper 25: Epistemic Border Control as Proxy Regulation Under Partial Observability

## Center of gravity

Projection / observability asymmetry / substitution under unsensed target variable. Where no real-time sensor on V exists and the available observables load on a proxy V′, the controller must substitute V′ for V; the rhetoric stays fixed on V. The substitution is structurally forced — sincere regulators facing the same sensor set produce the same substitution.

## Position in the series

P25 sits in the escalating-negative-results sequence P22→P26 as the *spatial* substitution result. Sibling to Paper 24 (aggregation-layer masking) by algebraic argument: aggregation improves SNR but does not rotate the observability subspace.
